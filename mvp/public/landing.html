Perguntas frequentes

Como funciona a PeopleFlow?

Você cria a conta, ativa os módulos que precisa (DP, RH, IA…) e convida o time. Colaboradores registram ponto e solicitam férias pelo portal; gestores aprovam; o DP acompanha tudo em dashboards — sem instalação, direto no navegador.

É multiempresa?

Sim. Cada empresa tem dados, usuários e permissões totalmente isolados — com isolamento reforçado no próprio banco de dados. Grupos podem administrar vários CNPJs em uma única conta.

Tem API?

Sim — API REST versionada (/api/v1) com documentação OpenAPI, API keys com escopos e webhooks para integrar com ERP, contabilidade e eSocial.

Possui IA?

Sim. O AI Engine responde dúvidas de CLT, gera contratos, advertências, descrições de cargos e resume currículos. Suporta OpenAI, Gemini, Claude e modelos locais — e nunca age sem confirmação humana.

É seguro?

LGPD by design: criptografia AES-256 para dados sensíveis, senhas com Argon2ID, MFA, permissões RBAC e trilha de auditoria imutável de todas as ações.
